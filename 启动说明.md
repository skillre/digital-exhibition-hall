# 数字展厅 - 启动说明

## ✅ 项目已就绪

我已经为你创建了完整的数字展厅项目，包含：

### 📁 项目文件
- **3D 场景**：Three.js 实现的沉浸式展厅空间
- **展板系统**：支持文档、图片、视频展示
- **漫游控制**：WASD 移动 + 鼠标视角
- **交互系统**：点击展板查看详情
- **UI 界面**：导航、传送、帮助等功能

### 🚀 快速启动

#### 方式一：本地预览（推荐）

由于不能在本地安装依赖，我们使用 Python 内置服务器：

```bash
# 进入项目目录
cd /Users/skillre/claude/系统开发/数字展厅

# 启动本地服务器
./scripts/serve.sh

# 或者直接运行 Python
python3 -m http.server 8000
```

然后在浏览器访问：http://localhost:8000

#### 方式二：VS Code Live Server

1. 用 VS Code 打开项目文件夹
2. 安装 "Live Server" 扩展
3. 右键点击 `index.html` → "Open with Live Server"

### 🌐 部署到云端（免费）

#### GitHub Pages 部署步骤

1. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 仓库名：`digital-exhibition-hall`
   - 选择 Public
   - 点击 "Create repository"

2. **推送代码**
   ```bash
   cd /Users/skillre/claude/系统开发/数字展厅
   
   # 初始化 Git
   git init
   git add .
   git commit -m "Initial commit"
   
   # 添加远程仓库（替换你的用户名）
   git remote add origin https://github.com/你的用户名/digital-exhibition-hall.git
   
   # 推送
   git branch -M main
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库 → Settings → Pages
   - Source: "Deploy from a branch"
   - Branch: "main" / "/ (root)"
   - 点击 Save

4. **访问你的展厅**
   ```
   https://你的用户名.github.io/digital-exhibition-hall/
   ```

### 🔧 自定义配置

修改 `js/main.js` 中的 `CONFIG` 对象：

```javascript
const CONFIG = {
  exhibition: {
    width: 40,    // 展厅宽度
    height: 8,    // 展厅高度
    depth: 40     // 展厅深度
  },
  player: {
    moveSpeed: 5,  // 移动速度
    height: 1.6    // 玩家高度
  }
};
```

### ❓ 常见问题

**Q: 页面空白怎么办？**
A: 检查浏览器控制台（F12）是否有错误，确保使用 Chrome 或 Firefox。

**Q: 点击展板没反应？**
A: 先点击画布锁定指针，然后才能点击展板。

**Q: 如何添加视频？**
A: 将视频文件放到 `content/videos/` 目录，然后在 `content.json` 中配置。
