# 数据安全服务数字展厅 - 开发指南

## 项目概述

这是一个基于 Three.js 的 3D 虚拟数字展厅，用于展示数据安全服务过程中产出的各类内容。

## 快速开始

### 1. 本地开发（不安装依赖）

由于不能在本地安装任何依赖，我们使用纯静态文件开发：

**方式一：使用 Python 内置服务器**
```bash
cd /Users/skillre/claude/系统开发/数字展厅
./scripts/serve.sh
```

**方式二：使用 VS Code Live Server**
1. 在 VS Code 中打开项目文件夹
2. 安装 "Live Server" 扩展
3. 右键点击 `index.html` -> "Open with Live Server"

**方式三：使用 Node.js（如果已安装）**
```bash
cd /Users/skillre/claude/系统开发/数字展厅
npx http-server -p 8000
```

然后访问 http://localhost:8000

### 2. 部署到 GitHub Pages

#### 步骤 1: 创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称：`digital-exhibition-hall`（或你喜欢的名称）
3. 选择 Public
4. 不要初始化 README（我们已经有了）
5. 点击 "Create repository"

#### 步骤 2: 推送代码到 GitHub

```bash
cd /Users/skillre/claude/系统开发/数字展厅

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 数据安全服务数字展厅"

# 添加远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/digital-exhibition-hall.git

# 推送
git push -u origin main
```

#### 步骤 3: 启用 GitHub Pages

1. 进入仓库设置页面
2. 点击左侧菜单的 "Pages"
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" 和 "/ (root)"
5. 点击 "Save"

等待几分钟后，你的展厅将可以通过以下地址访问：
```
https://你的用户名.github.io/digital-exhibition-hall/
```

## 添加新内容

### 1. 编辑内容配置

编辑 `content/content.json` 文件，添加新的展区和展板：

```json
{
  "exhibitions": [
    {
      "id": "new-exhibition",
      "name": "新展区名称",
      "description": "展区描述",
      "position": { "x": 0, "y": 0, "z": 0 },
      "panels": [
        {
          "id": "panel-new",
          "type": "document",
          "title": "展板标题",
          "description": "展板描述",
          "tags": ["标签1", "标签2"],
          "thumbnail": "assets/images/thumbnail.jpg",
          "contentUrl": "content/documents/document.pdf"
        }
      ]
    }
  ]
}
```

### 2. 内容类型

| 类型 | 说明 | 示例 |
|------|------|------|
| `document` | 文档类 | PDF、Word、PPT |
| `image` | 图片类 | JPG、PNG、SVG |
| `video` | 视频类 | MP4、WebM |
| `chart` | 图表类 | ECharts 配置 |

### 3. 添加静态资源

将图片、文档、视频等资源放到对应的目录：

```
content/
├── documents/   # 文档文件
├── images/      # 图片文件
└── videos/      # 视频文件
```

## 自定义配置

### 修改展厅尺寸

编辑 `js/main.js` 中的 `CONFIG.exhibition`：

```javascript
exhibition: {
  width: 40,    // 宽度
  height: 8,    // 高度
  depth: 40,    // 深度
  wallThickness: 0.3  // 墙壁厚度
}
```

### 修改灯光效果

编辑 `js/main.js` 中的 `CONFIG.lighting`：

```javascript
lighting: {
  ambient: {
    color: 0xffffff,
    intensity: 0.4
  },
  directional: {
    color: 0xffffff,
    intensity: 0.8,
    position: { x: 5, y: 10, z: 5 }
  }
}
```

### 修改玩家参数

编辑 `js/main.js` 中的 `CONFIG.player`：

```javascript
player: {
  moveSpeed: 5,      // 移动速度
  lookSpeed: 0.5,    // 视角速度
  height: 1.6,       // 玩家高度
  collisionDistance: 0.5  // 碰撞距离
}
```

## 常见问题

### Q: 3D 场景不显示？
A: 检查浏览器是否支持 WebGL，推荐使用 Chrome 或 Firefox。

### Q: 点击展板没有反应？
A: 确保已锁定指针（点击画布），然后才能点击展板。

### Q: 如何添加新的展区？
A: 编辑 `content/content.json`，添加新的展区配置。

### Q: 如何修改展板样式？
A: 编辑 `js/objects/ExhibitionHall.js` 中的材质配置。
